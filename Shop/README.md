# 🛒 Система управления магазином

## 📌 Описание
Учебный проект на Python, реализующий простую систему управления магазином.

## ⚙️ Возможности
- работа с товарами
- создание клиентов
- оформление заказов
- автоматический подсчёт стоимости заказа
- загрузка товаров из файла `products.txt`
